# 会話ログ保存コマンド
このコマンドはClaude Codeとの会話ログをマークダウン形式で保存します。

## 実行内容
1. 会話ログの整形
   - 現在のセッションでの全ての会話をマークダウン形式に整形
   - ユーザーとClaudeのやり取りを明確に区別
2. ファイルの保存
   - **Writeツールを使って**ファイルを保存すること
   - 保存先: `.claude/conversations/` ディレクトリ
   - ファイル名形式: `YYYY-MM-DD_<topic>.md`
   - トピック名は会話から自動推測して日本語で記載
   - ファイルパスの例: `/home/mo/runteq/NeuroWord/.claude/conversations/2026-01-23_トピック名.md`

## 出力フォーマット
```markdown
## まとめ
[会話の要約]

## 会話履歴

### User
[ユーザーの発言]

### Claude
[Claudeの回答]
```

## 注意事項
- **必ずWriteツールを使ってファイルを保存してください**
- マークダウンを出力するだけでなく、ファイルとして保存することが必須です
- .claude/conversationsのディレクトリ作成コマンドは実行不要です（既に存在しています）
- 本コマンド（/save-conversations）を実行するやりとりは会話履歴に含めないでください
- 保存完了後、保存したファイルパスをユーザーに伝えてください
